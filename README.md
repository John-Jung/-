## 💻프로젝트 소개

<p>SK 쉴더스 루키즈 19기 1차 모듈프로젝트입니다.</p>
<p>1달동안 배운 파이썬, Django, 애플리케이션 보안, 시스템/네트워크 보안 기술, 클라우드 보안 기술과정을 적용</p>
<p>SK 쉴더스취약점점검보고서 클라우드 보안 가이드, KISA 파이썬 시큐어코딩 가이드를 반영하여 개발</p>
